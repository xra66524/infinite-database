# 定位规则

## 定位目标

玩家知道一个目标的位置，称为定位目标。定位分为三种状态：精确定位、模糊定位、无法定位。

**精确定位**：精确定位指确切知道目标的位置。一般来说，对人类而言，如果能清楚明确且真实地看清目标，则可以定位它。人物可以无碍地攻击被定位的目标，或者对其使用那些“指定X个目标/指定范围内X个目标”的法术或能力。

**模糊定位**：大概知道目标的位置称为模糊定位。这意味着人物知道目标的大概方位，但不清楚具体位置。对人类来说，看不到人但能近距离听到持续的说话和嘈杂声，就有可能模糊定位。人物仍然可以攻击被模糊定位的目标，但成功数、附加成功和产生的效果点数都只计算一半，向下取整；人物也可以选择模糊定位的目标作为指定目标的法术或能力之目标，但法术只有一半概率成功。然而，如果攻击、法术或能力范围描述为“范围”或“范围内每个/所有单位”，则此攻击、法术或能力照常生效而无需承受减半或概率失手的后果。

**无法定位**：人物完全无法知道目标的位置。就人类而言，这意味着他看不到、听不到敌人，并且没有任何可供分析的数据；或者人物只知道其“大概所在的区域”。人物可以假定目标在某位置并进行攻击，若目标恰好在那个位置，则视为模糊定位。但人物不可能将他指定为法术或能力的目标。然而，如果攻击、法术或能力范围描述为“范围”或“范围内每个/所有单位”，则此攻击、法术或能力照常生效。当人物面对无法定位的目标的攻击时，视为措手不及（无支线）；若此效果是由于反侦测能力带来的，则根据3.4.3侦测与反侦测章节中的规则，可能会有更高等级的措手不及效果。

如无特别说明，对于人物敏感范围内未进行潜藏、也没有反侦测能力的目标，可以直接精确定位。

*当然，这是在人物的感官完好的前提下。更具体的感官与定位的关系见下一章**3.4.2感知方式**。

其他情况包括以下三种：

1、若目标具有反侦测能力，则需与对方进行侦测对抗，详见**3.4.3侦测与反侦测**。

2、若目标正在潜藏，则人物需要使用侦查检定，对抗对手的潜藏检定，详见潜藏技能的描述。只要你胜出即可模糊定位目标（一些特殊情况可能直接精确定位，例如对手并没有掩体，只是趁你注意力分散的时候从你身边走过等。由ST判断）；失败则意味着你无法定位目标，甚至没有察觉目标的存在。

3、对于敏感范围以外进行侦查时：

-每增加50米，侦查成功数-1。得到1成功数即可模糊定位，无法精确定位（仅对于没有任何特殊能力且没有躲藏的单位而言；如果对方具有特殊能力或者进行了躲藏，则结合规则1、2进行处理）

-侦查敏感范围以外的距离时，至少需要1轮时间；侦查的范围每增加敏感范围（或视觉范围，取决于玩家用什么方式侦查）的一半，总的侦查时间就会增加一倍；或者你也可以声明一定距离外等同你敏感范围（或视觉范围）大小的一块区域，你对这片区域的侦查只需要1轮时间，无论距离多远（但仍需承受侦查DP的减值）。通常来说，这适用于你使用望远镜等方式来进行侦测的情况。
