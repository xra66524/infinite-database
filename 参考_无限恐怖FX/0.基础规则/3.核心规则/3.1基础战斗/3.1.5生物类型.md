# 生物类型

## 生物类型

**黑暗生物**
这是个较为宽泛的概念。在无限世界中，所有在邪恶的魔法和超自然力量下诞生，或者血脉中含有这种力量的生物，都被视为黑暗生物。它包括但不限于不死生物、某些妖类和被特殊注明的生物。除了少数血统、修行和法术会给玩家带来这个特性之外，通常持有该特性的都是剧情生物，这一般需要特别给予标注。

**不死生物**
在无限恐怖的世界中遍布着各种不死生物，从DND世界到神话世界，特色各有不同，但如无特殊说明，它们在规则上都视为有以下特性

――没有恶性伤害槽。所有造成恶性伤害的特性对它们都造成严重伤害。只要被严重伤害填满生命槽，它们就立即被摧毁。

――无智慧的不死生物免疫心灵效果，有智慧的则不免疫。

――无法通过治疗药剂和神圣系恢复法术恢复生命。

――免疫无支线的毒素、药物、疾病影响。

――不需要呼吸、饮食和睡眠。

――有【光明】属性的武器会对其额外造成2点伤害。

**光明生物**
所有在善良的魔法和超自然的力量下诞生，或者血脉中含有这种力量的生物，都被视为光明生物。这一般是指天界生物，或者其他带有神圣血脉的生物。除了某些兑换会给玩家带来这个特性之外，通常持有该特性的都是剧情生物。

**虚体**
虚体指并没有实际形体，不会受到普通物理影响，也无法对其他事物施加任何物理影响，不能被闻到，听到，但却可以被看到，并且实际存在的事物。

虚体具有以下效果：

・虚体与虚体之间正常影响。

・虚体不会被普通物理行为影响或伤害到。但具有『魔法』『能量』『力场』『幽冥』等特性，以及能够正常影响灵体的行为都可以正常伤害或影响到虚体。当虚体进行这些类型的行为时，可以正常影响到现世。

・虚体不受重力影响也没有重量，可以以半速飞行，机动性为完美。

・虚体生物不受自然温度或现世普通物质影响，这意味着虚体生物可以穿透固体，液体等。但在固体中时虚体生物没有视觉。

・虚体生物故意的发声会被一般人听到。

**灵体**
灵体与虚体虽然十分相似，但从本质上说，灵体并不算实际存在的事物，或者说灵体并不属于这个世界或位面，灵体不但无法施加或受到物理影响，不能被闻到，听到，甚至无法被看到。

灵体具有以下效果：

・拥有『幽冥』特性的行为会直接对灵体造成恶性伤害并且正常影响灵体。除此之外如无特殊说明，可以正常影响虚体的特性在面对灵体时，有一半的概率伤害到灵体。

・和虚体一样，灵体不受自然温度或现世普通物质影响，可以以半速飞行，机动性为完美，正常穿透固体，液体等，固体中灵体没有视觉。

・灵体与灵体之间正常影响。灵体在进行拥有『幽冥』特性的行为时可以正常影响现世；在进行可以影响虚体的行为时，有一半概率影响现世。

・灵体发声除同属灵体外除非特殊说明否则无法听到。

**集群生物**
一个集群是一些超微型（真菌或细菌）、微型（蟑螂等昆虫）或超小型生物（老鼠、蝙蝠等）的集合，它们在行动方式上看作是一个生物。一个集群具有组成它的个体种类的性质。集群有单一的生命值，单一的先攻，单一的移动速度以及单一的防御等级。集群在进行豁免检定时亦和单个生物一样。

集群没有明显的前胸后背也不存在明确的生理结构，所以它不受致命攻击或类似的情况影响。

以单体攻击或射线类的攻击对集群效果较差，而范围类攻击对集群则有更好的效果，详见后文。

将一个集群的生命值减到没有完好或更低可以将它打散，但直到该数目以前的伤害都无法降低它攻击和抵御攻击的能力。集群不会因为受伤而失去战斗力或者处于濒死状态。它们无法被绊摔、擒抱或冲撞。

集群免疫针对具体数目生物的法术或效果，但如果集群具有智力值和群体意识，那么影响心灵的效果（魅惑、胁迫以及士气加值）是例外情况。影响区域的法术或效果可对群体额外造成一半的伤害。

此外，在擒抱检定中，群体在维持擒抱状态时仍然可以对其他目标进行攻击而不受减值。群体即使处于擒抱状态也不视为措手不及。在攻击时，群体通常的攻击方式为在其移动结束后，对所停留的空间内的生物进行攻击。群体攻击无视隐藏和掩蔽。正常计算伤害吸收和减免。

若群体溃败，视为群体中30%成员死亡，30%昏迷。对特定个体进行判断时投一次百分骰：结果为1-30时视为死亡，31-60时视为伤势严重导致的昏迷，61-100 时代表受害者逃脱了没有受伤。

当体型更大的单位组成群体时亦可被视为类集群生物（例如军队），由ST自行判断。

| 超小型（老鼠、蝙蝠） | -0.25 | -0.25 | +0.5 | -0.25 | 0 | 0 |
| 微型（蟑螂等昆虫） | -0.5 | -0.5 | +0.5 | -0.5 | 0 | 0 |
| 超微型（真菌或细菌） | -1 | -1 | +0.5 | -0.5 | -0.5 | 0 |

**在上表的“范围攻击”中，对于有数量限制的范围攻击（例如连射），如果攻击目标的数量超过了集群总数量的一半，则视为范围攻击处理；反之则视为单体攻击处理。

例如：

1、对超小型集群进行单体、造成穿刺伤害的攻击（例如长剑刺击），最终伤害倍率为-0.25-0.25=-0.5，也就是仅造成原本伤害的一半。

2、对微型集群进行锥形或圆形范围、造成穿刺伤害的攻击（例如火力压制），最终伤害倍率为+0.5-0.5=0，正常造成伤害

3、对超微型集群进行直线范围、造成纯能量伤害的攻击（例如带有超级贯穿的阳炎炮），最终倍率为-1+0=-1，也就是无法造成有效伤害
