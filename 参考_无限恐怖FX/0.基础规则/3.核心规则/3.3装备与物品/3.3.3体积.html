体积
体积定义的是一个生物（或物体），可以正常活动（对于物品来说，是使用）的空间大小。举例而言，尽管你可以将一个萝莉塞进旅行箱，但这也并不意味着这只萝莉就真的是体积2。它一般代表着该生物（或物品）的占地面积和高度。
因此，简单来说，体积就是“将一个生物或物品抽象成一个长方体后，其最长边的长度”。
对于人类，或者类人生物来说，其臂展与身高差距不高，只需要计算其身高就可以计算其体积。此时，每40cm（或者0.4米）为1体积。
体积1为0-40cm，体积2为41-80cm，体积3为81-120cm，体积4为121-160cm，体积5为161-200cm，以此类推。
武器大小与隐藏的关系：体积1=能藏在衣中；体积2=能藏在大衣中；体积3或更大=不能藏在身上。当人物体积更大或更小时，将这些数字进行等比例变化。

人物的体积改变
人物的体积值经常被称为体型。
一些能力或效果会使得人物的体型发生改变，这种超自然的变化会对人物产生肉体上的影响。更大的体型让人物更加强壮，更加坚韧，步幅更大，也更加容易被击中。

体型改变有如下效果：
・人物的基础速度取决于体型
・人物的基础生命值取决于体型。当人物由较大的体型变为较小的体型时，优先移除最接近完好的生命槽。
・人物的体型每比其基础体型（对玩家来说，5）提升2，就在力量相关的检定上获得+1DP加值，在满足装备所需力量时视为人物原本力量值+1。
・人物的体型每比其基础体型（对玩家来说，5）减小2，就在力量相关的检定上获得-1DP减值。在满足装备所需力量时视为人物原本力量值-1。
・人物在攻击与自己具有体型差距的单位时，目标体积每比自己高2点，他在受到攻击时就会在（闪避+格挡+掩蔽防御之和）上受到1点减值，按照闪避、格挡、掩蔽的顺序减少；目标体积每比自己低4点，你对其进行的单体攻击的攻击DP就会-1，但特殊的，这个减值可以被减少部位攻击减值的能力1:1抵销。
・一般情况下，人物身上穿戴、持用的物品会随着人物的体型改变而改变（除非能力特殊说明，否则改变体型的效果都会同时改变身上穿戴、持用的物品）。在这种情况下，人物的检定不会发生改变。注意，当人物的体形增大时，不要考虑武器随之变大带来的伤害改变，这被视为与体型增大而产生的命中减值相抵消。反之亦然。具有“格挡”特性的武器，也不会因为体积改变而带来更多或更少的格挡加值，因为其相对于人物本身的比例并没有改变。若无特殊说明，否则资源中的物品体积，无论是前置要求或者带来的效果，指的均是适用体积（见下文）。
・若人物身上穿戴、持用的物品不会随着人物的体型改变而改变，则该物品可能会失去作用，见下文。
选开规则：小之又小（点击展开）

这是一个默认关闭的选开规则。
大部分时候，我们将“体积1”视为体积的最小值即可。但是，如果有一天，你真的要深入鼠穴，或者在《蚁人》世界中与蚁人一同缩小到蚂蚁那么大，又或者需要跟替身一起去某不愿透露姓名的先生脑袋里探险怎么办？本细则即被用于解决此问题。

大部分时候，只要体积与场景是统一适配的，你无需做出改动，仍然将那些已经缩小的人“视为”体积5即可。如果你仍然希望将体积进行实际上的缩小，那么：
体积在1以下时，体积值每下降一半（计算小数点），就计算一次上文中“体积降低2点”带来的效果，即0.5、0.25……以此类推。举个例子，一个体积1（身高为0.4米）的角色被缩小到体积0.5（身高0.2米），他就在力量相关的检定上获得-1DP减值。在满足装备所需力量时视为人物原本力量值-1。同时，他在面对仍然是体积1的角色的时候，具有攻击DP+1的效果。

在此规则下，体积不高于1的角色在对体积5的目标攻击时，其力量值若低于其因为体型承受的力量检定减值，则一般都可以视为攻击无效，或者视为集群，由ST判断――用于避免强大的轮回者在一轮中遭受无数蚂蚁袭击而饮恨。



物品体积的改变

・人物适用的物品体积与人物的体型比例是同步的。例如，5体型的人物适用1体积的匕首，10体型的人物适用2体积的匕首，以此类推。
・近战武器的触及范围和所需力量取决于其本身的体积值。
・当你持用的武器的体积每比你适用的体积提升1，视为力量需求+1；体积每减小1，视为力量需求-1，最少到0。
・当你持用的武器（或工具）的体积每比你适用的体积提升或降低2，你在使用此武器或者工具进行检定的时候就需要承受-2DP器械减值，与不满足力量需求所需承受的减值叠加；同时，在使用该武器进行格挡格挡动作时，获得的格挡加值也会-2。
・防具提供的防御加值不会随着其体积的增大而提高或降低。但穿着不合身的盔甲时，盔甲的标准适用体积值每提高或降低2，盔甲减值就会提升2点。

无论是武器、工具、盔甲还是盾牌，当因为不合适的大小带来的减值与不满足力量需求带来的减值达到6，即可判定人物根本无法以此物品的原本使用方式使用此装备。此时，该装备无法为人物提供任何加成。
举例而言，一个力量1的玩家【无法以此物品的原本使用方式来使用】一面体积4的风筝盾（来自于力量需求的器械减值达到6）；一个力量3、体积3的侏儒，也【无法以此物品的原本使用方式来使用】这面适用于体积5的人物的风筝盾（来自力量需求的器械减值为4，同时来自体积不合导致的格挡减值为2）。
细则：强行使用不合适的武器（点击展开）

当一件武器已经被判断为“无法以此物品的原本使用方式来使用”（或者玩家认为通过新方式使用武器更有利）的时候，人物可通过以下两种方式使用该武器：
1、用作临时武器，见上一章节。
2、将该武器作为玩家适用体积另一种相同体积、类别相近的武器使用，此时，它会失去原本的特性，将其视为一把无支线的新武器。例如，玩家彭三缴获了一把体积4的C级匕首，具有：武器伤害6L、破甲5、高速4；彭三没有武器，他选择将其视为一把无支线的重剑（体积4）来使用。
无论是哪一种方式，新的使用方式下，若无特殊说明，原本武器的绝大多数特性都会失去作用。但在ST判断中，或许原本的武器也许有一些特性仍然可以继续发挥作用，例如，只要携带就能生效的效果；或者武器表面存在一种只要受伤就能致命的强力毒素，诸如此类。



人物的触及
人物的基础触及一般为体积（身长/臂展）/2+所持武器的体积（长度）。
你需要将体积按照体积规则转化为米计算，例如一个1米8（体积5）的人，手持一把1米（体积3）的长剑，此时他的基础触及=1.8/2+1=1.9米（这是基于现实数据的运算，你无需在计算中舍去小数点）。



购买大体积的装备
一般来说，兑换获得的装备总是与你的体积适应。但你也可以主动兑换更大或更小体积的物品，这不需要额外的分数。这意味着你可以给战斗机器人直接兑换武器，而无需进行数值上的调整。
