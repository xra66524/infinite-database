# 前述

这是以2.5规则（神猪整理版本，《无限恐怖2.5PHBV0.3.chm》）为主干，增加了部分新二版，以及蛮多改动的规则环境。其中较大的区别包括以下几点：

-取消了2.5的大能规则，取而代之的是2.0为基准的附加成功规则，并有一定权重和效果修改。

-完全开放的撰写规则。以特性位和特性DP值为基准，并根据等级具有一定加成。

-简化/修改了某些规则，如造物、敏感范围、载具驾驶、硬度等。

-不再是只有有限的血统和强化提供属性加值，而是大量资源根据等级直接提供属性值和属性值上限。

-区分购买获得的属性和强化获得的属性，只根据购买的属性、而非当前属性值来计算以分数进行属性强化的花费。

更多和更为具体的修改内容请自行查看。

资源大多来自于《无限恐怖强化资源.chm》、纯美苹果园、好腿，及其他各种各样的地方。后期有很多小伙伴投稿，多谢多谢。

这个环境是一个较弱、对能力影响范围做出极大限制、而且禁止很多高收益搭配的规则环境。相对来讲，我们希望ST有更安心于讲一个好故事的开团环境――比如，不会在低级出现什么影响全世界的致死气候；没有无限DP；只要把属性和技能点到上限，PC的数值很容易拉平……诸如此类。
