# 无神祭司胸衣

 B+4000

魔幻本质

这件胸衣的样式与批评求知欲妓院的失宠女士所穿的样式非常相似，有着整洁漂亮的外观，并且可以完美地展现穿戴者的性感身材――如果穿戴者的身材确实如失宠女士一般性感的话。印记城裁缝冈尔卡夫在为这件独特衣服的编织施以魔法时花了很大心思。它不仅可以清洁自己，用自己的织物修补小裂口，还可以使穿戴者的心智更加昌明。

无神祭司胸衣视作轻甲，具有盔甲防御8/8，穿戴者在盔甲防御上获得+5附加成功，并获得6点全能量抗力。

此外，若购买者具有任何魔幻来源的通用能量池（比如：魔力），则在购买时绑定其中一个通用能量池，每次长休息后，胸衣会自动生成30点该能量池类型的能量，以供穿戴者消耗。在每次长休息开始时，若胸衣生成的能量仍未消耗完，则该些能量立刻消散。
