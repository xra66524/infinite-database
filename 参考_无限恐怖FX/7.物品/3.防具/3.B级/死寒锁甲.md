# 死寒锁甲

 B+2500

魔幻本质

　　这件锁甲并非以俗世的金属作为材料而制成的。传说，制造死寒锁甲的材料，必须等到极夜，北极最寒冷的时刻。将冬夜的精华萃取，并注入纯净的十六面水晶体中。接着在长达一年的时间中，不断的用魔法对水晶加以强化，稳定其形态，并将其雕琢成宛若雾气的丝状。而要制作一件死寒锁甲，需要岂止成千上万条这样的丝线。

　　最终成型的死寒锁甲看上去仿佛是一件轻薄的丝衣，摸上去仿佛并不存在一般。然而即使是再尖锐的刃器也难以穿透这层若有若无的防御。

    这是一件轻甲，盔甲防御为12/12，『防弹』，『防能量武器』。穿戴者获得3点寒冷抗力和3点火焰抗力，以及DR3/魔法。

    另外，每场战斗一次，死寒锁甲的穿戴者可以消耗一个标准动作并支付一点意志力，在瞬间抽取空气中的热量，对以穿戴者为中心15米半径的范围造成20点寒冷严重伤害，受影响的生物以强韧豁免抵消伤害，并且每受到1点伤害，就会受到1点冻结点数（标准豁免）。
