# “变形法袍”

魔幻本质

B+2000

介绍：

这是一件古代魔法帝国风格的长袍。

“三级中阶魔法袍，恒定‘外观变形能力’，并提供三环魔法师的魔法抗力与二级骑士的防御能力，同时每天可以变形为老鼠、猫头鹰、巨魔各一次。”

效果：

盔甲防御：12/12，全能量抗力3点，占据轻甲位

‘外观变形’：你可以自由变换这件长袍的款式和颜色

每天可以自由动作变形为老鼠、猫头鹰、巨魔各一次，持续1小时。这是等同于你施法者等级的变形来源的能力。可以自由动作提前解除。

老鼠：你的体积变成1，其他数据不变。

猫头鹰：你的体积变成1，获得等于基础速度的飞行速度，机动性普通。

巨魔：你的体积提高3点（计算生命值时按原体积算），获得快速医疗1【医疗点12】，受到酸性和火焰伤害时，这份快速医疗失效一轮。

在衣服中留下精神记印记后，可以接收到制作者留下的信息：

“变化的是外形，不变的是内心。――阿松桑・菲利普”
