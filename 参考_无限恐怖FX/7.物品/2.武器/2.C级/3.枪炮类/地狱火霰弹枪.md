# 地狱火霰弹枪

价格：C+1000

“英国佬看上去很有把握”

“英雄永远是英雄，加布里尔”

“雕像上那个英雄可不是我”

这个后来被称为死神的人，本名加布里尔・莱耶斯。他出生于洛杉矶。智械危机开始前，他成为了美国军队的高级军官，是一名久经沙场、受人敬佩的老兵。因技战术精湛后加入了实验性小队守望先锋，并被给予了这把地狱火霰弹枪。在随后升职？黑色守望队长期间，手下遗传学家莫伊拉对其进行了改造，使他获得了某些超乎常理的能力（无论是地狱火霰弹枪还是布里尔・莱耶斯本人）。

本质：特异本质

制造商：守望先锋

类型：霰弹枪

体积：2

弹药类型：无限

装弹量：4 发（不可拆卸管状弹仓）

操作原理：泵动式

基础射程：10米

力量需求：4（单手）

武器伤害：10L

效果：

-地狱火霰弹枪的弹道扩散很大，因此，其有效射程要比一般的枪械近得多，每超出一倍基础射程后，武器伤害下降1L，在远距离上将几乎不产生任何伤害。不过，因其较大的扩散范围，换来的是对极近距离内单位的伤害，当对距离你不超过1米的目标射击时，地狱火霰弹枪武器伤害提高5L。

-特殊的，在购买本武器后，你会获得无数把完全一致的地狱火霰弹枪，但这些地狱火霰弹枪都视为处于一个特殊的储物空间中，只有你可以从中取出和使用，他人无法使用该枪械，并且枪械在离开你手一轮后也会自动消失。你可以同时拔出两把地狱火霰弹枪，每一把枪械的体积为2，取出和存入是一个移动动作，如同拔出随身携带的武器（存取两把或者一把武器动作相同，可适用于即时备战专长或类似效果），重新取出的地狱火霰弹枪总视为满备弹状态。但你只需要支付正常的价格就可以将武器改造的相关效果作用在所有的“地狱火霰弹枪”上。
