# 对魔用特种手枪

-Jackal CC+1500

魔幻本质

这是一把通体漆黑，刻着银色符文的怪物手枪，它的口径为13mm，全长39cm，重19kg。这并不是设计用来对人战斗，而是专门用来对付魔怪的――对付人类，根本用不到到如此可怕的武器。

武器伤害12L，射程60米，弹数10+1，单手持，力量需求6，体积1，专业：手枪，具有破甲3、破魔2和【眩晕】特性。

这是一把使用专门弹的枪械，不适用基础手枪弹。专门弹分为以下两种：

1、专用13mm穿甲弹，弹壳为银制特殊合金，内装药为马贝尔斯化学药筒NNNA9，弹头是经过洗礼仪式的水银弹头。若使用此子弹，Jackal的攻击将获得【光明】和【幽冥】特性，并视作具有【银】材质。价格为5分1发。

2、没有经过洗礼的特殊银合金弹，若使用此子弹，Jackal的攻击视作具有【银】材质。价格为1分10发。

变体：撒玛利亚人，巨大的左轮手枪，弹数减至4发，失去【眩晕】特性，但会获得1点攻击上的附加成功（器械加值）。其余数据相同。
