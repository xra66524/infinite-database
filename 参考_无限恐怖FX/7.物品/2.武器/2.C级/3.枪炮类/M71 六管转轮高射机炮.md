# M71 六管转轮高射机炮

M71 Anti-Aircraft Gun俗称Scythe，因其巨大的尺寸和沉重的质量，让UNSC大多时候将其作为一种载具或者地面上的固定式旋转机炮炮塔，虽然有着极限情况下的单兵使用的记录，但即使穿着雷神动力甲的二代斯巴达，将其作为单兵武器都显得捉襟见肘，步履蹒跚。

它作为是一种强大的导气式风冷武器，配备有100,000,0发的供弹具，发射二十一世纪最离谱的电影导演也不敢采用的单兵武器设定：口径惊人的20x102㎜高爆穿甲弹；拥有六根枪管的它，可以以每分钟6,000发的速度开火，毫无疑问，没有任何碳基生物可以抵挡如此强大的火力。

基础射程：150米

持用：双手/支架固定

特性：

【转轮机炮】

出于该武器的结构限制，一旦开火，武器仅可以进行连射或者火力压制。

【有效电机】

有效电机在启动后，炮管只需极小的旋转时间便可开火，并伴随着发出响亮高昂的嗡嗡开火声 ，如同骤雨般密集的金属风暴对目标区域将造成毁灭性的打击，将一切生命体和轻装甲撕成碎片。

【M715 20x102㎜  AP弹药】

M71 六管转轮机炮拥有惊人的供弹具，该供弹具在满载情况下，重量到达了10公吨，高度更是到达了3米，即使在极端情况下，M71 六管转轮机炮作为单兵武器使用也并不会携带该供弹具，但机炮本身上百发的弹容，即使在有意降低射击速度的前提下，也仅可以支撑几秒的射击，因此在主神空间出售的M71 六管转轮机被直接赋予了【无限子弹】特性。而适配武器本身的高爆穿甲弹【AP弹药】，使得该武器在射击时，视为具有【破甲6】

【钛合金炮盾】

M71 六管转轮机炮枪管后部附近装有一个可伸缩的金属护罩，以保护机枪手 ，这种装甲使用和星际战舰装甲同样的材质：A钛合金，操作者可以在开启射击时同步展开护盾，金属护盾将为机枪手提供3点额外的盔甲防御。
