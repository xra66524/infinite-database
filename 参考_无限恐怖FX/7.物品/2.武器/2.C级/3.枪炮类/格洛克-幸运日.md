# 格洛克-幸运日

（Lucky Day） C+1500

从外观上这只是一把9mm格洛克17型手枪，只是枪托上刻着一个七孔的马蹄铁。

手枪数据如下：武器伤害为5L，射程为60米，弹数17+1,其他数据和普通大型自动手枪相同。

它还具有以下特性：

1、上帝的硬币：

当你需要进行机运骰或基因锁解锁检定时，你可以在1和2两个数字中选择，选择后投掷一次d2。如果d2的结果与你选择的结果相同，那么这次机运骰或基因锁解锁检定结果+2；失败则结果-1，除非你投掷的数字是10。这个d2不受任何效果影响。

2、今天是我的幸运日，不是你的幸运日：

当你使用这把左轮进行攻击时，你可以忽略攻击目标身上最多6点防御上的机运加值。

3、虽然上面没写着你的名字，但这发子弹注定是你的：

每次你进行攻击时，你都可以投掷一次d10，如果这次d10出10，你可以忽略这次攻击目标的基本防御和闪避防御，这是一个C级命运来源的能力。这个效果成功触发后，一场景内不能再触发第二次。这个d10不受任何效果影响。
