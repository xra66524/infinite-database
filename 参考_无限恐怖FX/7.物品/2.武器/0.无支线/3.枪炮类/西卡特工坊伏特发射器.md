# 西卡特工坊伏特发射器

 Sicatt Workshop Volt Caster

本质：科技本质

价格：500

西卡特工坊伏特发射器 ，俗称为干扰器手枪，是一种低噪音的潜入式电能弧击武器，可以让敌方步兵或载具触电，并释放高输出电流。

类型：手枪

制造商：西卡特工坊

长度：64.9㎝ （25.5英寸，体积2），重量：2.1kg ，力量需求：1

弹药类型：电激光（造成电击伤害）

武器伤害：4L

基础射程：30米

射击模式：半自动

西卡特工坊伏特发射器/干扰器手枪,外形为银白色外表的大型手枪，在枪管的前三分之一处，是由四个导轨和一个电弧发生器组成的伏特发射器，而在一般枪械滑膛的位置，则被一个自持续型的高能条状电池组成，在开枪时，伏特发射器与枪管链接处将开始旋转，并发出淡蓝色的光。

自持续电池为枪械提供的电能可视为无限，但伏特发射器只可连续进行10次射击，这视为拥有10发子弹，连射将消耗对应的子弹数，当连续射击10次后，需要以移动动作手动拉开电池进行散热，才可继续射击。

当使用西卡特工坊伏特发射器对由电能作为能源（或存在电路板）的科技本质物品进行攻击时，这次攻击获得+1dp增强加值，且伤害无需减半。当枪械造成的伤害大于等于其结构值时，目标视作因为短路而陷入瘫痪。
